Try it out

Clone Tsatsiki,

git clone git://github.com/boblail/tsatsiki.git
cd tsatsiki

Install the bundle,

bundle install

Run migrations and seed the site with the Project data for Tsatsiki,

rake db:create db:migrate db:fixtures:load
