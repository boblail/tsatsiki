Tsatsiki is a Project Management tool that uses Cucumber Features for tickets.

Getting Started

Check out the repo:

git clone git://github.com/boblail/tsatsiki.git
cd tsatsiki

Install the bundle:

bundle install

Run migrations and seed the site with an administrator account and the Project data for Tsatsiki:

rake db:setup

Configuring Tsatsiki

In order for Devise to send a Password Reset email and for Devise Invitable to send invitations, Tsatsiki needs to know how to send emails from your computer.

To configure Tsatsiki, rename the email.yml.example file to email.yml and set its properties as you would ActionMailer’s.

Finally run Tsatsiki. You can either create a new account or log in with the username admin and the password password.
